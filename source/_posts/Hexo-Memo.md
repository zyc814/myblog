## Writting

1. New
```python
hexo new [post/page/draft] "title"
hexo publish [post/page/draft] "title"
```

2. Generate 
```python
hexo g  //generate
hexo s  //server
```

3. Deploy
```python
hexo d //deploy
```

